# NTMT-Network-Traffic-Monitoring-Tool-
NTMT — асинхронный сетевой прокси и монитор трафика на языке Rust.
Утилита перехватывает исходящие и входящие соединения, логирует заголовки и тело запросов/ответов в файл, а также перенаправляет трафик на целевые серверы. Предназначена для отладки сетевых приложений, анализа трафика.
Ключевые возможности:
- Прозрачная маршрутизация HTTP-запросов
- Асинхронное логирование трафика в бинарном/текстовом виде
- Неблокирующая архитектура на базе Tokio
- Просмотр сырых байтов запросов и ответов для низкоуровневой отладки
В данной версии присутствует только HTTP-маршрутизация. В последкющем будт добавлена HTTPS и остальные протоколы. Ожидайте.
